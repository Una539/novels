# 终焉与始源

> *"终结不是毁灭，而是守护的开始。"*

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **类型** | 少年成长 × 浪漫冒险 × 政治斗争 |
| **篇幅** | 长篇（360章，三卷） |
| **核心主题** | 在终结与新生之间，选择守护而非毁灭 |
| **结局** | 王思衡升格为终焉，希莉丝升格为始源，保留人性和意识，以非人类形态守护世界流明平衡 |

---

## 三卷结构总览

| 卷 | 章节 | 核心任务 | 主线敌人 |
|---|---|---|---|
| **第一卷：暗潮涌动** | 1-120 | 日常→相遇→卷入兄弟会阴谋→清剿兄弟会 | 血晶兄弟会 |
| **第二卷：铁幕之下** | 121-240 | 铁幕真面目→对抗→重组→联邦篇→救下艾拉→铁幕收容冲突 | 铁幕（腐败派系） |
| **第三卷：终焉黎明** | 241-360 | 教会真相→艾拉失控→最终仪式→升格→新世界 | 终末教会 |
